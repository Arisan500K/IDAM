# Sistem Arisan 500K

Proyek ini adalah sistem arisan online yang dibuat untuk mengelola arisan bulanan Rp500.000.

## Fitur

- Login anggota
- Sistem pembayaran
- Spin arisan (Roda / Botol / Dadu)
- Demo spin
- Timer arisan
- Admin panel
- Sistem anti manipulasi
- Riwayat pemenang

## Teknologi

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express

Database:
- Firebase / Supabase

## Struktur Proyek

frontend → tampilan website  
backend → server logika arisan  
database → struktur data  
docs → dokumentasi sistem
